## Welcome to Asteroids!

Asteroids! is a 2D bullet hell lateral scroll created with Unity.

### Game

When you execute the game you will appear in the initial screen and you have to press the button "Play" to start. If you die you can restart the game or go back to the main menu.

### Controls

 * A W S D/Arrows/Left joystick - Movement
 * Space/A button - Use primary slot inventory
 * Alt-Left/B button - Use secondary slot inventory
